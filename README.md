## 🌟 Overview

2. Historical Context & Logic

## 1. The Geneva Mechanism 

This Mechanism is unique because its utility has evolved significantly over centuries:

*  **Origin**: It was designed by watchmakers to limit the number of rotations the spring of the watch can go through, thus preventing the spring from being wound too tight.. This was done by blocking or making one of the slots shorter as to limit the drive wheel to go only through a set number of rotations. This design was called a "Geneva Stop".
*  **The Cinematic Era**: Then it was used, as the same design used in this project, in film projectors. The mechanism allowed each frame to remain stationary in front of the lens for a fraction of a second.
*  **Modern Day**: Even with the presence of electronic systems and servo motors, the Geneva Drive remains a staple in mechanical indexing due to its high reliability and dwell-time precision.

## 2. The Indexing Turntable

To demonstrate practical application, I simulated a **Rotary Indexing Table** for a bottling line. This one of the applications where the **Geneva Drive** still dominates even with modern technology. The Design was Done using **SOLIDWORKS**

## 3. Design and CAD modeling
### Key Technical Specifications for the Geneva Drive:
*    Mechanism Type: 4-Slot External Geneva Drive.
*    Motion Ratio: 1:4 (90° rotation per drive wheel revolution).
*    Dwell Time: Designed for a 3:1 dwell-to-motion ratio.

### Key Technical Specifications for the TurnTable:
Accordingly the turntable was designed with 4 stations. This is a typical layout for a bottling line.
* Station 1 (Loading): Conveyor belt feed.
* Station 2 (Processing): Filling operation during the "dwell" phase.
* Station 3 (Assembly): Capping operation.
* Station 4 (Offloading): Discharge to output conveyor.
